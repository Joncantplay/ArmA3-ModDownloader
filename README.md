# ArmA3 ModDownloader
Updating/Installing ArmA3 Mods via HTML file
## Features

- ArmA 3 Server Update
- Mod Download via HTML file
- converting upercases in lowercase (for Linux)
- Copying keys to the key folder
- Print mods launchparameter
- Log file
## Usage

```sh
python3 a3down.py path/to/html

Choose a option:
1. Full Update (With ArmA Server) # Updating/downloading ArmA3 Server and mods
2. Normal Update # Only updating/downloading mods
3. Only print Launch options + copy keys
4. Exit


## Run it if you have missing dependencies ##
python3 a3down.py setup
```
